# Tableau Dashboard

This dashboard was created in Tableau Public to analyze ecommerce cosmetics product data.

## Dashboard Link

[View the Tableau Dashboard](https://public.tableau.com/views/CosmeticProduct_17747939551790/Main?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

## Dashboard Focus

The dashboard includes analysis of:

- Product categories and subcategories
- Brand distribution
- Product prices
- Customer ratings
- Website and country-based product availability

## Tools Used

- Tableau Public
- SQL
- Python
- GitHub
